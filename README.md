Installation
=========

1. Clone this repository
2. `touch config1.js`
3. Open up config.js, copy the below template, and fill in with your Twitter developer credentials.
4. `npm install`.


> exports.twitterAccess = {

>  "consumer_key": '',

>  "consumer_secret": '',

> "access_token": '',

> "access_token_secret": ''

> };

> exports.screen_name = "";
